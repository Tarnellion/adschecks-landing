Scope definition

What we verify

The public scope of AdsChecks is intentionally narrow: one verified page, selected context, and evidence outputs.

Unit
One completed page load in a selected GEO/device context.
Slot limit
Up to 5 predefined ad slots per verified page.
Outputs
Statuses, screenshots, JSON metadata, UTC timestamp.

AdsChecks is an audit and verification service.

Verified page unit

A Verified page means one completed page load for a target URL in a selected check context (GEO + device) and verification of up to 5 predefined ad slots.

Delivered evidence

Each verified page result includes:

  • slot statuses
  • screenshots
  • JSON metadata
  • UTC timestamp

Service boundary

AdsChecks provides audit, evidence, and verification.

AdsChecks does not guarantee ad delivery, ad availability, impressions, or revenue outcomes.
